Project Overview

SmartCare was developed to solve the chaos of manual ICT maintenance tracking. It provided a centralized platform where technicians and administrators manage the entire lifecycle of hardware repairs and service requests.

Key Workflows

  • Role-Based Access: Separate dashboards for technicians and administrators.
  • Admin Approval: Admin authorizes new users and equipment additions.
  • Technician Search: Search for existing devices before creating new logs.
  • Lifecycle Management: Track devices from "Pending" to "Approved".
  • Secure Backend: Password hashing with relational MySQL database.

Technologies Used
